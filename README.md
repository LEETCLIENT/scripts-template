# [RU] Шаблон для папки скриптов Leet Client

Базовая структура для папки скриптов. Рекомендуется использовать Visual Studio Code для разработки скриптов.

Данный шаблон даст tab complete по структуре скрипт апи.

Для классов net.minecraft документация [здесь](https://maven.fabricmc.net/docs/yarn-1.21.8+build.1/index.html).

# [EN] Leet Client scripts template

Basic structure for scripts folder. It's recommended to use Visual Studio Code or similar for script development.

This template gives tab complete for the script API structure.

Read the docs [here](https://maven.fabricmc.net/docs/yarn-1.21.8+build.1/index.html) for net.minecraft classes.
